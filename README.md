Data Analytics Projects showcasing customer behaviour analysis using python , sql , power bi.
## üìå Project Overview
- This project focuses on analyzing customer shopping behavior for a retail business using Python, SQL, and Power BI. ## üìÇ Dataset Description

- Records: 3,900 purchases
- Features: 18 columns
- Key Data Points:
  - Customer demographics (Age, Gender, Location, Subscription Status)
  - Purchase details (Category, Item, Amount, Season, Size, Color)
  - Behavioral metrics (Discount usage, Reviews, Purchase frequency, Shipping type)
  - Missing Values: 37 values in Review Rating (handled during data cleaning)

 --- 

## üîß Tools & Technologies

- Python: Pandas, NumPy, Matplotlib (EDA & data cleaning)
- SQL: PostgreSQL (business queries & customer segmentation)
- Power BI: Interactive dashboard & visual storytelling

  ---

## üßπ Data Preparation & Analysis

#### Python (EDA & Cleaning)
- Cleaned and standardized column names
- Handled missing review ratings using median imputation by category
- Performed feature engineering (age groups, purchase frequency)
- Prepared data for SQL analysis

#### SQL (Business Analysis)
- Key insights extracted using SQL queries:
- Revenue comparison by gender
- Subscriber vs non-subscriber spending behavior
- High-spending discount users
- Customer segmentation (New, Returning, Loyal)
- Top products by rating and category
- Revenue contribution by age group and shipping type

  ---

## üìä Power BI Dashboard

- An interactive dashboard was created to visualize:
- Revenue trends by demographics
- Subscription impact on revenue
- Shipping type vs purchase amount
- Product performance and customer segments
- The dashboard enables stakeholders to quickly identify high-value customers and growth opportunities.
